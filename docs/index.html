WHAT IT DOES

Counts confirmed watch time on short vertical video in Chromium. Not tab time. Not background noise. Only when the clip is actually playing in focus.

Native mobile apps are not tracked.

INSTALL

  1. Download the .zip (234 KB)
  2. Unzip the archive on your computer.
  3. Open chrome://extensions in Chrome, Edge, or Brave.
  4. Enable Developer mode (top right).
  5. Click Load unpacked and select the unzipped folder.
  6. Pin Scroll Receipt. Open a Reel, Short, or TikTok tab to start.
