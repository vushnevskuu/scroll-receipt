# Chrome Web Store listing (draft)

**Name:** Scroll Receipt

**Short description:** Daily email receipts for short-form video time in your browser.

**Permissions justification:**
- `storage` — local daily totals and settings
- `tabs` — detect active tab for accurate playback tracking
- `idle` — pause counting when you're away
- `alarms` — finalize sessions and sync schedule
- Host access to instagram.com, youtube.com, tiktok.com — detect Reels/Shorts playback only

**Not tracked:** native mobile apps, messages, comments, page content, full browsing history.
