SETUP RECEIPT

INSTALL

~2 MIN · CHROME / EDGE / BRAVE

  1. Download the .zip (242 KB)
  2. Unzip on your computer.
  3. Open chrome://extensions in Chrome, Edge, or Brave.
  4. Turn on Developer mode.
  5. Click Load unpacked → select the unzipped folder.
  6. Pin Scroll Receipt. Watch a Reel or Short to see your first totals.
  7. Open Options → enter your email → verify the code to enable daily receipts.
