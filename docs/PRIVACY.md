# Privacy

## Principle

**Your activity stays on this device.**

Scroll Receipt measures active short-form viewing on supported websites. It does not read messages, comments, captions, searches, passwords, or unrelated browsing activity.

## What we collect

- Active viewing duration on YouTube Shorts, Instagram Reels, TikTok
- Content engagement metrics (viewed, engaged, completed, quick skips)
- Session timestamps and platform totals
- User settings (platform toggles, equivalent rates, receipt time)

## What we never collect

- Messages, comments, captions, searches, typed text
- Screenshots or screen recordings
- Video downloads
- Unrelated website activity
- Raw page URLs (only hashed content identifiers)

## Storage

All data is stored in `chrome.storage.local` on your device. No cloud sync in MVP. No analytics SDKs. No tracking pixels.

## Controls

- Pause tracking
- Disable individual platforms
- Export data as JSON
- Delete today's data
- Delete all data

## Scope label

Receipts are labeled **BROWSER ACTIVITY ONLY** — native mobile apps are not tracked.
